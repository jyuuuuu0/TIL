# SSG (Static Site Generation)

SSG는 빌드 시점에 HTML을 미리 생성해 두고 사용자에게 제공하는 방식이다.

즉, 페이지를 미리 만들어 두었다가 요청이 오면 바로 전달한다.

대표적으로 Next.js, Gatsby에서 사용된다.

## 동작 과정

1. 빌드 단계에서 HTML 파일 생성
2. 생성된 HTML을 서버 또는 CDM에 저장
3. 사용자가 요청하면 즉시 HTML 제공

## 특징

### 장점

- 매우 빠른 로딩 속도
- 서버 부하가 거의 없음
- SEO 매우 유리

### 단점

- 데이터가 자주 변경되는 사이트에는 부적합
- 변경 시 다시 빌드 필요

### 적합한 상황

- 내용이 자주 변경되지 않는 팀 소개, 서비스 소개, FAQ와 같은 사이트.
